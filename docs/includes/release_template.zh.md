### 更新方式

和以往一样，可根据安装方式选择以下任一命令：

#### docker-compose

```bash
docker-compose pull
docker-compose up -d
```

#### setup 脚本安装

```
# 请先关闭虚拟环境，然后执行
./setup.sh --update
```

#### 原生安装

```
git pull
pip install -U -r requirements.txt
```
