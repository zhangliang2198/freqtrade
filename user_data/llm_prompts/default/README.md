# 📚 LLM 决策提示词 - 社区版 vs 知识星球版

## 🎯 关于此目录

此 `example` 目录包含的是 **简化社区版本** 的 LLM 决策提示词模板。这些模板可以让你快速上手体验 AI 驱动的交易决策系统，但功能相对基础。

---

## 🆓 免费版特点

### ✅ 包含功能
- ✅ 基础的角色定位和决策框架
- ✅ 基本的技术指标显示
- ✅ 简单的资金和持仓状态
- ✅ 核心决策逻辑
- ✅ JSON 格式输出

### ⚠️ 功能限制
- ❌ 无详细的决策优先级框架
- ❌ 无原始 K线数据分析
- ❌ 无多周期技术指标
- ❌ 无风险指标（止损距离、清算距离）
- ❌ 无调仓历史记录
- ❌ 无常见错误警示系统
- ❌ 无高级策略指导

**Token 消耗**: 约 200-400 tokens/次决策

## 💎 知识星球版本 (default/aggressive/conservative)

### 🚀 核心优势

#### 1️⃣ **显著提升决策质量**

付费版本提供了 **系统化的决策框架**，而不是让 AI 盲目猜测：

| 决策点 | 免费版 | 付费版 |
|--------|--------|--------|
| **入场决策** | 3条基本原则 | **4个优先级** + 风险回报比分析 + 市场环境评估 |
| **出场决策** | 3条基本原则 | **6个优先级** + 调仓历史评估 + 利润保护策略 |
| **调仓决策** | 3条基本原则 | **6个优先级** + 做T策略 + 仓位硬性约束 |
| **仓位管理** | 3条基本原则 | **5个决策因子** + 波动率动态调整 + 额度计算 |
| **杠杆管理** | 3条基本原则 | **5个决策因子** + 清算风险评估 + 保守/激进策略 |

#### 2️⃣ **关键风险指标**

这是付费版本的 **杀手级功能** - 实时风险监控：

```
## 风险指标
- 止损价: $95.000000 | 距止损: 5.00%
- 初始止损: $95.000000 (5.00%)
- 跟踪止损: 已启用
- 清算价: $90.000000 | 距清算: 10.00%
- 有效止损/清算价: $95.000000
```

**为什么重要？**
- 🚨 **避免爆仓**: 杠杆交易中，清算距离 < 5% 时 AI 会强烈建议减仓
- 💰 **保护利润**: 距止损 < 3% 时 AI 会主动考虑平仓
- 📊 **精准评估**: AI 能准确判断当前风险水平，而不是凭感觉

**免费版没有这些数据，AI 可能会在接近清算时还建议加仓！**

#### 3️⃣ **完整的市场数据**

付费版提供多维度的市场洞察：

**多周期技术指标**:
```
## 技术指标 (5m)
- RSI: 65.23
- MACD: 0.0023
- EMA_20: 42350.45

## 参考指标 (15m)
- RSI: 58.67
- MACD: 0.0015
```

**原始 K线数据** (最近 32 根):
```
- 2025-11-14 10:00: {'open': 42300, 'high': 42450, 'low': 42250, 'close': 42400, 'volume': 1250}
- 2025-11-14 10:05: {'open': 42400, 'high': 42500, 'low': 42380, 'close': 42480, 'volume': 1180}
```

**为什么重要？**
- 📈 AI 可以识别趋势（连续 5 根阳线 = 强势上涨）
- 🔍 AI 可以发现形态（双底、头肩顶等）
- ⚡ AI 可以分析多周期共振（5m + 15m + 1h 同时看涨）

**免费版只有单周期指标值，AI 无法做深度分析！**

#### 4️⃣ **调仓历史追踪**

付费版记录每一次仓位调整：

```
## 近期调仓记录
1. 15分钟前 | ADD @ $42300 | 投入$500 | limit
2. 45分钟前 | REDUCE @ $42500 | 投入$300 | limit
3. 2小时前 | ADD @ $42100 | 投入$400 | limit

⚠️ 关键: 根据上述调仓记录，避免重复操作！
```

**为什么重要？**
- 🔄 **避免频繁操作**: AI 能看到刚做过加仓，不会立即再加
- 📊 **评估策略效果**: AI 能分析做T是否降低了成本
- 🎯 **优化决策**: AI 能根据历史调整判断当前策略是否有效

**免费版没有历史记录，AI 可能会连续多次相同操作！**

#### 5️⃣ **常见错误警示系统**

付费版内置了 **10+ 种常见错误警示**，教育 AI 避免典型失误：

**调仓决策中的警示**:
```
## 常见错误（务必避免）
❌ 最严重错误: 减仓导致剩余仓位低于最小开单额度
❌ 忽视仓位计算: 减仓前不计算剩余金额
❌ 忽视调仓历史: 不看近期操作记录，重复相同操作
❌ 操作过于频繁: 短时间内反复调仓
❌ 下跌趋势中盲目加仓摊平（无技术支撑确认）
❌ 资金紧张或调整次数用尽时强行加仓
❌ 盈利仓位遇阻力不减仓（错失做T机会）
```

**为什么重要？**
- 🎓 **AI 学习**: 这些是真实交易中血的教训
- 🛡️ **风险保护**: 显著降低错误决策概率
- 💡 **决策质量**: AI 会主动检查是否犯了这些错误

**免费版没有警示系统，AI 可能重复犯经典错误！**

#### 6️⃣ **精细化策略配置**

付费版提供 **3种风格** 的提示词：

| 风格 | 特点 | 适用场景 |
|------|------|----------|
| **default** | 平衡风险收益 | 大多数交易者 |
| **aggressive** | 激进进取，追求高收益 | 高风险偏好者 |
| **conservative** | 保守稳健，严格风控 | 风险厌恶者 |

**例如调仓策略差异**:
- **Aggressive**: 允许逆势加仓摊平成本（有条件）
- **Conservative**: 严格禁止给亏损仓位加仓
- **Default**: 平衡，根据技术面决定

**免费版只有一种风格，无法个性化！**
