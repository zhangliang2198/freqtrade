# LLM 提示词说明

## 概述

本目录包含了用于 Freqtrade LLM 策略的提示词模板。这些模板用于指导 AI 模型做出交易决策。

## 提示词版本

### 免费版

当前提供的是简化版提示词，包含基本的交易决策框架。这些提示词可以让你体验 AI 交易的基本功能。

### 知识星球版

完整的提示词模板需要加入知识星球获取。完整版包含：

- 更详细的决策框架
- 风险管理指标
- 多周期技术分析
- 调仓历史追踪
- 常见错误警示系统
- 三种不同风格配置（保守、默认、激进）

## 自定义提示词

所有源码中可用的变量均已提供，您可以根据变量说明文档轻松构建自己的完整提示词模板。详细变量说明请参考 `CONTEXT_VARIABLES.md` 文件。

## 提示词风格

系统支持三种不同的提示词风格：

- **conservative**: 保守型，风险控制优先
- **default**: 默认型，平衡风险与收益
- **aggressive**: 激进型，追求高收益

详细风格配置说明请参考 `README_PROMPT_STYLES.md` 文件。

## 使用方法

1. 在配置文件中设置 `prompt_style` 参数选择风格
2. 系统会自动加载对应风格的提示词模板
3. 如需自定义，可参考现有模板和变量说明进行修改

## 注意事项

- 修改提示词可能会影响交易决策质量
- 建议在回测环境中测试新的提示词
- 完整版提示词提供更专业的决策框架，建议加入知识星球获取
